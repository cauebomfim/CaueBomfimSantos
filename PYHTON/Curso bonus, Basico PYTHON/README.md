# Exercícios - Python Básico 🐍

Exercícios realizados durante um curso introdutório de Python, já concluído, com foco nos conceitos fundamentais da linguagem e na prática da lógica de programação.

## 📚 Conteúdo das aulas

1. Imprimindo "Olá Mundo"  
2. Primeiras variáveis  
3. Primeiras operações matemáticas  
4. Interação com o usuário (`input`)  
5. Operações matemáticas com interação do usuário  
6. Primeira lista (array)  
7. Imprimindo o primeiro e o último elemento da lista  
8. Alterando e adicionando elementos na lista  
9. Removendo elementos da lista (`remove()` e `del`)  
10. Utilizando `for loop`  
11. Utilizando `for loop` com `range()`  
12. `for loop` alinhado (Nested loop)  
13. Utilizando `while`  
14. Uso de `break` e `continue` em loops  
15. Contagem de itens em uma lista  

## 📌 Observação

Este material representa a prática dos conceitos iniciais da linguagem Python durante a realização do curso.
